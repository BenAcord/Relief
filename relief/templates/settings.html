Settings

Presentation File Name Prefix

The default file name prefix is relief and will be used to create a file named relief-TrafficLight-CAMPAIGNNAME-YYYYmmdd-HHMM. Where CAMPAINNAME is the selected campaign or current campaign to export. If a different prefix is entered below it will replace the "relief" portion of the name. For example, enter your company name for branding. Additionally, this setting is used as the title property (see below).

Traffic Light Default

Similar to data marking and document handling the traffic light protocol is a four color-coded matrix for information sharing. For further information please see NIST special publication 800-150. In short summary, RED = no sharing, AMBER = organizational limited disclosure, GREEN = community limited disclosure, and WHITE = no limitations. Relief's default value is AMBER. All slides will be marked with the traffic light indicator. The export filename will also include the traffic light indicator.

Data Marking Footer

Not all reports are intended for the public. By entering a company specific marking each slide footer will be marked accordingly. Common markings may include: "Internal use only", "Restricted", "Company Confidential". By default, a generic "For Internal Use Only" marking is applied.
